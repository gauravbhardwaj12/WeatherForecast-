# Weather App 🌤️

A simple weather app built with HTML, CSS, and JavaScript.
https://github.com/gauravbhardwaj12/WeatherForecast-

## Features
- Search weather by city
- Shows temperature, humidity, and condition
- Last 5 search history

## How to Use
1. Enter a city name in the search box
2. Click the **Search** button
3. View the weather details

## API Used
- [WeatherAPI](https://www.weatherapi.com/)

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Enter your API key in `script.js`
```javascript
   const apiKey = 'YOUR_API_KEY';
```
